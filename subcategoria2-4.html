¿QUÉ ES?

Un filtro SVG en HTML es una forma de aplicar efectos gráficos a los elementos gráficos (como imágenes, formas y texto) mediante el uso del elemento 'filter' dentro del lenguaje SVG (Scalable Vector Graphics). Los filtros SVG permiten la manipulación de los gráficos de manera avanzada, permitiendo crear efectos como desenfoque, sombras, cambios de color, y muchos otros.

Elementos

1.Desenfoque (blur): Este filtro difumina los bordes de un elemento, creando un efecto de desenfoque.

2. Brillo (brightness): Este filtro ajusta el brillo general de un elemento. Un valor mayor que 1 lo hace más brillante, mientras que un valor menor que 1 lo hace más oscuro.

3. Contraste (contrast): Este filtro ajusta la intensidad de los colores en un elemento. Un valor mayor que 1 aumenta el contraste, mientras que un valor menor que 1 lo disminuye.

4. Sombra (drop-shadow): Este filtro agrega una sombra a un elemento, simulando el efecto de la luz que incide sobre él.

5. Escala de grises (grayscale): Este filtro convierte un elemento a escala de grises, eliminando toda la información de color.

6. Manipular el color (hue-rotate): Este filtro gira el espectro de colores de un elemento, creando un efecto de cambio de tono.

7. Negativo (invert): Este filtro invierte los colores de un elemento, intercambiando los claros por los oscuros y viceversa.

8. Opacidad (opacity): Este filtro ajusta la transparencia de un elemento. Un valor de 0 lo hace completamente transparente, mientras que un valor de 1 lo hace completamente opaco.

9. Saturación (saturate): Este filtro ajusta la intensidad de los colores en un elemento. Un valor mayor que 1 aumenta la saturación, mientras que un valor menor que 1 la disminuye.

10. Sepia: Este filtro da a un elemento un tono sepia, similar al de las fotografías antiguas.

Código de Ejemplo:

                
   <style>
    svg {
      border: 1px solid black;
      background-color: lightgrey;
      margin: 20px;
    }
    .blur {
      filter: blur(5px);
    }
    .brightness {
      filter: brightness(1.5);
    }
    .contrast {
      filter: contrast(200%);
    }
    .drop-shadow {
      filter: drop-shadow(10px 10px 5px black);
    }
    .grayscale {
      filter: grayscale(100%);
    }
    .hue-rotate {
      filter: hue-rotate(90deg);
    }
    .invert {
      filter: invert(100%);
    }
    .opacity {
      filter: opacity(50%);
    }
    .saturate {
      filter: saturate(300%);
    }
    .sepia {
      filter: sepia(100%);
    }
