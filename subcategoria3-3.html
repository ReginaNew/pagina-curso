¿QUÉ ES?

Campos Requeridos, Deshabilitados y de Lectura en HTML En los formularios HTML, existen tres atributos que permiten controlar la interacción del usuario con los campos de entrada: required, disabled y readonly.

Required

El atributo required es un atributo booleano. Cuando lo usamos, especifica que se debe completar un campo de entrada antes de enviar el formulario, haciendo el campo obligatorio. El atributo funciona con los siguientes tipos de entrada: texto, búsqueda, URL, teléfono, correo electrónico, contraseña, selectores de fecha, número, casilla de verificación, radio y archivo.

Disabled

El atributo disabled es un atributo booleano. Cuando lo usamos, especifica que el campo de entrada, esta deshabilitado, es decir inutilizable. Este atributo permite que el usuario utilice el campo, hasta que se complan algunas condiciones.

Readonly

El atributo readonly es un atributo booleano. Cuando lo usamos, especifica que el campo de entrada, será de solo lectura. Un campo de entrada de solo lectura no se puede modificar (El usuario puede tabular y copiar el texto).

Código de Ejemplo Required:

                
    <form action="action_page.php" method="post" id="myForm">
    <label for="name">Nombre:</label>
    <input type="text" id="name" name="name" required>
    <br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    <br>
    <button type="submit">Enviar</button>
</form>

Código de Ejemplo Disabled:

                
    <label for="name">Nombre:</label>
        <input type="text" id="name" name="name" value="Regina" disabled>
<br>
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" value="regina@gmail.com" disabled>
    </form>

Código de Ejemplo Readonly:

                
    <form action="action_page.php" method="post">
        <label for="name">Nombre:</label>
        <input type="text" id="name" name="name" value="Regina" readonly>
<br>
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" value="regina@gmail.com" readonly>
    </form>
