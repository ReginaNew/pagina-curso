¿QUÉ ES?

El elemento "canvas" en HTML se utiliza para dibujar gráficos y otras representaciones visuales mediante JavaScript. Este elemento proporciona un área sobre la que se puede dibujar gráficos 2D o 3D.

Elementos:

1. Coordenadas: El elemento canvas tiene coordenadas X e Y, donde X es la distancia en el eje horizontal y la Y es la vertical.
2. Formas y Colores: El elemento CANVA dispone de diversas propiedades y funciones que nos permiten dibujar formas como: rectángulos, círculos, curvas, líneas, texto, sombra, etc.
3. Colores: El color de los elementos dibujados no lo especificamos en la función, toma el color que se tenga configurado en ese momento como color de relleno, líneas, etc. El color puede ser determinado por un valor RGB, Hexadecimal o por nombre. Por ejemplo:
nombre = "Orange"
Hexadecimal ='#FFA500
RGB ="rgb(255, 165, 0)"

Elementos para el Estilo

1. fillRect(): Esta función sirve para dibujar rectángulos rellenos de color. Recibe cuatro parámetros, con este formato: fillRect(x,y,anchura,altura). Esto dibuja un rectángulo cuya esquina superior izquierda está en el punto (x,y) y con dimensiones de altura x anchura.
2. fillStyle: Especifica el color de relleno, de un objeto rectángulo en el canvas.
ctx.fillStyle="Orange"
ctx.fillStyle ='#FFA500'
ctx.fillStyle ="rgb(255, 165, 0)"
3. strokeRect: Esta función sirve para dibujar simplemente la silueta de un rectángulo, es decir, sólo su borde. Recibe cuatro parámetros.
strokeRect(x,y,anchura,altura)

4. strokeStyle: Utilizamos la propiedad strokeStyle para asignar el valor RGB que deseemos para el borde de los rectángulos.
ctx.fillStyle= "LightGreen"
ctx.fillStyle = '#90EE90'
ctx.fillStyle = "rgb(144, 238, 144)"
5. clearRect(): Esta función nos sirve para borrar áreas rectangulares de un canvas y hacerlas totalmente transparentes o sin contenido gráfico.
clearRect(x,y,anchura,altura)

Código de Ejemplo:

                
            
const canvas = document.getElementById('myCanvas');
const context = canvas.getContext('2d');

// Dibujar la cara (círculo grande)
context.beginPath();
context.arc(250, 250, 200, 0, 2 * Math.PI);
context.fillStyle = 'yellow';
context.fill();
context.stroke();

// Dibujar el ojo izquierdo
context.beginPath();
context.arc(170, 180, 30, 0, 2 * Math.PI);
context.fillStyle = 'black';
context.fill();

// Dibujar el ojo derecho
context.beginPath();
context.arc(330, 180, 30, 0, 2 * Math.PI);
context.fillStyle = 'black';
context.fill();

// Dibujar la boca (arco)
context.beginPath();
context.arc(250, 300, 100, 0, Math.PI, false);
context.lineWidth = 10;
context.strokeStyle = 'black';
context.stroke();
