¿QUÉ ES?

En el desarrollo web, un Grid Layout, también conocido como Diseño de Cuadrícula o CSS Grid, es un sistema de diseño bidimensional que permite organizar y posicionar elementos de forma flexible y eficiente dentro de una página web. Una página web dinámica, es aquella que se adapta a diferentes tamaños de pantalla, existen herramientas que nos permiten que las páginas respondan a ciertos factores generalmente el ancho de pantalla del dispositivo y la resolución de la pantalla, sin tener que sin recargar el contenido.

Grid View

Es una técnica que implica el diseño de 12 columnas, con esto podemos ajustar los contenidos de acuerdo al ancho de la pantalla, lo que facilita la colocación de elementos en la pantalla.

Background-size

Es una propiedad de CSS que especifica como un navegador web debe mostrar una imagen como fondo para una página web. Dependiendo de la calidad y del estilo de la imagen se puede establecer cómo queremos que esta se muestre.

Componentes Principales

-Contenedor de la cuadrícula: Es el elemento HTML que define el área donde se aplicará el diseño de cuadrícula.exclamation Se identifica con la propiedad display: grid o display: inline-grid.
-Líneas y columnas: Dividen el contenedor en celdas. Se definen con las propiedades grid-template-rows y grid-template-columns.
-Áreas de cuadrícula: Son las secciones nombradas que se crean dentro de la cuadrícula para colocar elementos específicos. Se definen con la propiedad grid-template-areas.
-Ubicación de elementos: Los elementos HTML se posicionan dentro de las celdas o áreas de la cuadrícula usando las propiedades grid-row-start, grid-row-end, grid-column-start y grid-column-end.

Código de Ejemplo:

                
    < /* Estilos CSS */
    .contenedor {
      display: grid; /* Activar el diseño de cuadrícula */
      grid-template-columns: repeat(3, 1fr); /* Definir 3 columnas de igual tamaño */
      grid-gap: 20px; /* Espacio entre celdas */
      padding: 20px;
      border: 1px solid #ccc;
    }

    .item {
      background-color: #eee;
      border: 1px solid #ccc;
      text-align: center;
      padding: 10px;
    }
   </style>
 </head>
 <body>
   <div class="contenedor">
     <div class="item">Elemento 1 </div>
     <div class="item">Elemento 2 </div>
     <div class="item">Elemento 3 </div>
     <div class="item">Elemento 4 </div>
     <div class="item">Elemento 5 </div>
     <div class="item">Elemento 6 </div>
   </div>
 </body>
