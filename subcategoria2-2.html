¿QUÉ ES?

En HTML, los arcos y los círculos se dibujan principalmente utilizando el elemento "canvas" en combinación con JavaScript. El "canvas" es un lienzo que permite a los desarrolladores dibujar gráficos dinámicos y complejos, como líneas, rectángulos, círculos, y arcos, utilizando una API de dibujo en 2D proporcionada por el contexto del canvas.

Arco

Para dibujar un arco utilizamos el método arc(x, y, r, ap, af, aC), donde
- x, y: son las coordenadas del centro
- r: especifica el radio del círculo
- ap: es el ángulo de inicio (en radianes)
- af: es el ángulo final (en radianes)
- cR (contra reloj) es la dirección.

Circulo

Para dibujar un círculo hay que trazar un arco de 360° ( 2π radianes ), Utilizamos el método arc().
ctx.arc(X,Y,r,0,2*Math.PI);

Código de Ejemplo:

                
    const canvas = document.getElementById('myCanvas');
    const context = canvas.getContext('2d');

    // Dibujar un círculo completo
    context.beginPath();
    context.arc(100, 100, 50, 0, 2 * Math.PI);
    context.fillStyle = 'blue';
    context.fill();
    context.stroke();

    // Dibujar un arco (medio círculo)
    context.beginPath();
    context.arc(300, 100, 50, 0, Math.PI);
    context.strokeStyle = 'red';
    context.lineWidth = 5;
    context.stroke();

    // Dibujar un arco (cuarto de círculo)
    context.beginPath();
    context.arc(500, 100, 50, 0, Math.PI / 2);
    context.strokeStyle = 'green';
    context.lineWidth = 5;
    context.stroke();

    // Dibujar un círculo con solo borde
    context.beginPath();
    context.arc(100, 300, 50, 0, 2 * Math.PI);
    context.strokeStyle = 'purple';
    context.lineWidth = 5;
    context.stroke();

    // Dibujar un arco (tres cuartos de círculo)
    context.beginPath();
    context.arc(300, 300, 50, 0, 1.5 * Math.PI);
    context.strokeStyle = 'orange';
    context.lineWidth = 5;
    context.stroke();
    
    // Dibujar un arco (más de un círculo completo)
    context.beginPath();
    context.arc(500, 300, 50, 0, 2.5 * Math.PI);
    context.strokeStyle = 'pink';
    context.lineWidth = 5;
    context.stroke();
