¿QUÉ ES?

En HTML y CSS, el desbordamiento de contenido ocurre cuando el contenido de un elemento supera las dimensiones de su contenedor. Esto puede suceder por varias razones como:
Texto demasiado largo: Si el texto de un elemento no cabe dentro del ancho del contenedor, se desbordará hacia los lados.
Imágenes grandes: Si una imagen es más grande que el contenedor, se desbordará por los bordes.
Márgenes y relleno: Si se aplican márgenes o relleno al elemento, estos aumentan su tamaño total, lo que puede provocar un desbordamiento si el contenido ya es grande.
Posicionamiento: Si el elemento está posicionado de forma absoluta o relativa, puede desbordarse del contenedor normal del flujo de documentos.

Overflow

CSS nos permite especificar como nuestro navegador debe manejar el contenido que desborde (exceda) su elemento. La propiedad especifica si se recorta el contenido o se agregar barras de desplazamiento cuando el contenido de un elemento es demasiado grande para caber en el área especificada. Solo funciona para elementos de bloque con una altura especificada.

Propiedades de Overflow

visible- Por defecto. El desbordamiento no se recorta. El contenido se muestra fuera de la caja del elemento.
hidden- El desbordamiento se recorta y el resto del contenido será invisible.
scroll- El desbordamiento se recorta y se agrega una barra de desplazamiento para ver el resto del contenido.
auto- Similar a scroll, pero agrega barras de desplazamiento solo cuando es necesario

Propiedades CSS

word-break: Controla cómo se rompen las palabras largas en varias líneas.
text-overflow: Controla cómo se muestra el texto que no cabe dentro del contenedor.
box-sizing: Controla cómo se calcula el tamaño total de un elemento, incluyendo márgenes y relleno.

Código de Ejemplo:

                
    <style>
        body {
            font-family: Arial, sans-serif;
        }
        .container1 {
            width: 300px;
            height: 200px;
            border: 1px solid black;
            margin-bottom: 20px;
        }
        .overflow-scroll {
            overflow: scroll;
        }
        .overflow-visible {
            overflow: visible;
        }
        .overflow-auto {
            overflow: auto;
        }
        .content1 {
            width: 500px;
            height: 300px;
            background-color: lightblue;
        }
    </style>
