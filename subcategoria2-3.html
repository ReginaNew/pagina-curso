¿QUÉ ES?

SVG significa Scalable Vector Graphics (Gráficos vectoriales escalables). Con HTML5 es posible insertar etiquetas SVG en HTML, lo que permite crear dibujos vectoriales directamente en la web: círculos, líneas, arcos, rutas, cuadros, polígonos, texto e imágenes gráficas. Para comenzar a dibujar utilizaremos la etiqueta 'svg'. Es necesario definir las dimensiones a la etiqueta (width y height).

Elementos

1. Coordenadas: Utiliza un sistema de coordenadas (x,y) para el dibujo de los elementos.

2. Lineas: Las líneas se definen con la etiqueta 'line'. Para definirla tenemos que indicar las coordenadas del punto inicial y final (x1 y1 x2 y2), en pixeles o porcentaje. Si de definen en pixeles no se añade “px” al valor. Para que la muestre tenemos que asignarle un color “stroke”.

3. Circulos: Los círculos se definen con la etiqueta 'circle'. Para este elemento debemos introducir el centro del círculo y su radio (cx cy r). Podemos personalizar el contorno con los mismos parámetros “stroke” empleados para la línea. Para el relleno utilizaremos la propiedad “fill” Para cambiar la opacidad utilizamos “fill-opacity” , con valores entre 0 y 1. Si no se definen parametros, el círculo será color negro sin borde.

4. Poligonos: Un polígono es un elemento que contiene al menos tres lados. Los polígonos se forman con líneas rectas y la forma es "cerrada" es decir todas las líneas se conectan. Se utiliza la etiqueta 'polygon' Los puntos del polígono se introducen todos con la propiedad “points”

Código de Ejemplo:

                
    <svg width="400" height="400">
                
                <rect x="100" y="100" width="100" height="100" fill="blue" stroke="black" stroke-width="2" >
                
                <polygon points="100,100 150,50 250,50 200,100" fill="lightblue" stroke="black" stroke-width="2" >
                
                <polygon points="200,100 250,50 250,150 200,200" fill="darkblue" stroke="black" stroke-width="2" >
                
                <line x1="100" y1="100" x2="150" y2="50" stroke="black" stroke-width="2" >
                <line x1="200" y1="100" x2="250" y2="50" stroke="black" stroke-width="2" >
                <line x1="100" y1="200" x2="150" y2="150" stroke="black" stroke-width="2" >
                <line x1="200" y1="200" x2="250" y2="150" stroke="black" stroke-width="2" >
                <line x1="150" y1="50" x2="150" y2="150" stroke="black" stroke-width="2" >
                <line x1="250" y1="50" x2="250" y2="150" stroke="black" stroke-width="2" >
            </svg>
